## Introduction

npmq is a lightweight wrapper around the process of quickly starting a new Node.js 
project, typically meant to be published on npmjs.com.

## Features

npmq offers some handy features and shortcuts:

- Simple CLI-based interactive configuration
- Unlimited configurable package templates so you can quickly start projects of any type
- Automatically sets up GitHub repositories with your desired settings
- Automatically sets up a package.json, LICENSE, and README.md file according to your preferences
- Free cloud sync for your npmq settings and package templates
